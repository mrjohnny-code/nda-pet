# NDA - pet проект

Интерфейс системы управления структурированными данными с фильтрацией, сортировкой и навигацией между сущностями.  
Реализованы роутинг, управление состоянием UI и работа с иерархической структурой данных (папки/элементы).

Изначально интерфейс был реализован на Nunjucks, SCSS и JavaScript в рамках реального проекта, после чего адаптирован и перенесён на React для портфолио с сохранением ключевой логики и структуры, а также частичным скрытием функционала и данных в соответствии с NDA.
